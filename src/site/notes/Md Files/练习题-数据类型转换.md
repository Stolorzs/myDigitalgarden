## 答案
```java
//1、
short s = 12;  // 对 12在范围之内
s = s-9;    //错  int不可赋给short  左边的s为short
//2、
byte b = 10; //对
b = b + 11; // 错  把int 赋给了byte
b = (byte)(b + 11);   //对 强转
//3、
char c = `a`; //对c = 'a'
int i = 16; // 对
float d = .314F;  // 对
double result = c +　ｉ　+ d; //对自动转换float，再计算，可以自动转换为double
//4、
byte b = 16;  // 对
short s = 14; // 对
short t = s + b;  //错 int不能自动转换为short
```
- 1、错
- 2、错
- 3、对 
- 4、不对
