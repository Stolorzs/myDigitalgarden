#### 程序流程控制介绍
- 在程序中，程序运行的流程控制决定程序是如何执行的，是我们必须掌握的，主要有三大控制语句
	- 1、顺序控制
	- 2、分支控制
	- 3、循环控制
- --
#### 顺序控制
- 程序从上到下逐行执行，中间没有判断跳转
- Java中定义变量采用合法的**向前引用** 
```java
public class Test{
	int num1 = 12;
	int num2 = num + 2; //上下顺序不可颠倒
}
```
