- **介绍**：赋值运算符就是将某个运算后值，赋给指定的变量
- **赋值运算符的分类**：
	- 基本赋值运算符`"="` 
	- 复合赋值运算符
		- `+=`：`a += b` --> `a = a + b`
		- `-=`：`a -= b` --> `a = a - b`
		- `*=`
		- `/=`
		- `%=`
- **案例**
```java
int n1 = 10;
n1 += 4;
System.out.println(n1);//14
n1 /= 3;
System.out.println(n1);//4
```
---
##### 赋值运算符的特点
- 1、顺序从右向左 
- 2、左边只能是变量，右边可以是变量、表达式、常量值
- 3、复合赋值运算符
- 4、复合赋值运算符会进行类型强制转换
```java
byte b = 3;
b += 2; //b = b + 2; 右边精度会提升为int,按理说无法赋给byte类型的b，但复合运算符会转换
//真正等价于 
b = (byte)(b + 2);
//同意
b++;//真正等价于
b =(byte)(b + 1);
```
