|       | 运算符优先级排序，向下依次递减                      |                         |
| ----- | ------------------------------------ | ----------------------- |
|       | `.` `()` `{}` `;` `,`                |                         |
| L<--R | `++` `--` `~` `!(data type)`         | **单目运行** ``++`` ``--``  |
|       | **算数运算符**                            |                         |
| L-->R | `*` `/` `%`                          |                         |
| L-->R | `+` `-`                              |                         |
|       | **位移运算符**                            |                         |
| L-->R | `<<` `>>` `>>>` `位移`                 |                         |
|       | **比较运算符**                            |                         |
| L-->R | `<` `>` `<=` `>=` `instanceof`       |                         |
| L-->R | ==   !=                              |                         |
|       | **逻辑运算符**                            |                         |
| L-->R | `&`                                  |                         |
| L-->R | `^`                                  |                         |
| L-->R | `｜`                                  |                         |
| L-->R | `&&`                                 |                         |
| L-->R | `｜｜`                                 |                         |
|       | **三元运算符**                            |                         |
| L-->R | `?`  `:`                             |                         |
|       | **赋值运算符**                            |                         |
| L<--R | ` =`   `*=`    `/=`     `%=`         |                         |
|       | `+=`    `-=`    `<<=`      `>>=`     |                         |
|       | `>>>=`      `&=`      `^=`      `｜=` |                         |
|       |                                      |                         |
